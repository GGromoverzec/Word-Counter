# WordCounter

WordCounter — простая программа для подсчета заданных голосовых фраз через микрофон.

Программа слушает речь, распознает текст и увеличивает счетчик, если услышана одна из указанных фраз. Подходит для стримов, игр, локального учета событий и любых ситуаций, где нужно считать повторяющиеся голосовые команды.

## Возможности

- Распознавание речи через микрофон
- Настраиваемый список фраз для подсчета
- Выбор языка распознавания: русский, украинский, английский
- Выбор языка интерфейса: украинский, русский, английский
- Выбор микрофона в настройках
- Отображение только активных микрофонов из настроек Windows
- Лог распознанных фраз
- Ручной сброс счетчика
- Ручная установка значения счетчика
- Настраиваемый текст счетчика
- Оверлей-режим
- Автоматическое сохранение настроек
- Автоматическая установка в AppData при первом запуске
- Создание ярлыка на рабочем столе
- Сборка в один `.exe` файл
- Иконка приложения

## Где хранятся настройки

Начиная с новых версий настройки хранятся в папке пользователя:

`AppData\Local\WordCounter\settings.json`

Это позволяет не засорять папку с программой и проще обновлять приложение.

## Запуск

Скачайте и запустите:

`WordCounter_1.4.2.exe`

При первом запуске программа автоматически создаст рабочую папку в `AppData\Local\WordCounter` и ярлык на рабочем столе.
